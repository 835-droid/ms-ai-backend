# تحديث: فصل ملفات JavaScript

## ما تم عمله

تم فصل ملف `app.js` الكبير إلى ملفات JavaScript منفصلة لكل صفحة لتحسين:

- **الأداء**: تحميل أسرع للصفحات
- **الصيانة**: كود أسهل في الفهم والتعديل
- **التنظيم**: فصل الاهتمامات بوضوح

## هيكل الملفات الجديد

```
cmd/web/
├── common.js          # الوظائف المشتركة (المصادقة، التخزين، المساعدات)
├── auth.js            # وظائف صفحة تسجيل الدخول والتسجيل
├── dashboard.js       # وظائف مكتبة المانجا
├── control.js         # وظائف لوحة التحكم
├── admin.js           # وظائف لوحة الإدارة
├── manga-details.js   # وظائف صفحة تفاصيل المانجا
└── app.js             # الملف القديم (محفوظ للرجوع إليه)
```

## الملفات المُحدثة

### common.js
- `showMessage()` - عرض الرسائل
- `showNotification()` - التنبيهات الصغيرة
- `saveTokens()` / `getAccessToken()` - إدارة الرموز
- `checkAuthentication()` - التحقق من المصادقة
- `handleLogout()` - تسجيل الخروج
- `handleApiError()` - معالجة الأخطاء
- `uploadImage()` - رفع الصور (مؤقت)

### auth.js
- `handleLogin()` - معالج تسجيل الدخول
- `handleSignup()` - معالج التسجيل
- `toggleAuthForms()` - تبديل النماذج
- `setupAuthListeners()` - إعداد event listeners

### dashboard.js
- `loadMangas()` - تحميل المانجا
- `displayMangas()` - عرض المانجا في الشبكة
- `searchMangas()` - البحث في المانجا

### control.js
- `checkAdminStatus()` - التحقق من صلاحيات المسؤول
- `updateUIForPermissions()` - تحديث الواجهة حسب الصلاحيات
- `loadQuickStats()` - تحميل الإحصائيات السريعة

### admin.js
- `switchTab()` - تبديل التبويبات
- `addManga()` - إضافة مانجا جديدة
- `addChapter()` - إضافة فصل جديد
- `loadMangaList()` - تحميل قائمة المانجا
- إدارة العلامات والنماذج

### manga-details.js
- `loadMangaDetails()` - تحميل تفاصيل المانجا
- `loadMangaChapters()` - تحميل فصول المانجا
- `displayMangaDetails()` - عرض تفاصيل المانجا
- `displayMangaChapters()` - عرض الفصول
- `openChapter()` - فتح الفصل للقراءة

## الصفحات المُحدثة

تم تحديث جميع الصفحات لتشمل الملفات المناسبة:

- **index.html**: `common.js` + `auth.js`
- **dashboard.html**: `common.js` + `dashboard.js`
- **control.html**: `common.js` + `control.js`
- **admin.html**: `common.js` + `admin.js`
- **manga-details.html**: `common.js` + `manga-details.js`

## المزايا

### ⚡ أداء أفضل
- تحميل أسرع للصفحات
- تقليل حجم JavaScript لكل صفحة
- تحميل المحتوى عند الحاجة فقط

### 🔧 صيانة أسهل
- فصل واضح للوظائف
- كود أسهل في القراءة والتعديل
- إمكانية إعادة استخدام الوظائف المشتركة

### 🐛 تصحيح أسهل
- أخطاء معزولة لكل صفحة
- تتبع أسهل للمشاكل
- اختبار أسهل للوظائف

### 📦 قابلية التوسع
- إضافة صفحات جديدة بسهولة
- فصل الاهتمامات بوضوح
- كود منظم وقابل للصيانة

## كيفية الاستخدام

1. **للصفحات الجديدة**: أنشئ ملف JS منفصل وأضفه إلى HTML
2. **للوظائف المشتركة**: أضفها إلى `common.js`
3. **للوظائف الخاصة**: أضفها إلى ملف الصفحة المناسب
