# إعادة تصميم الداشبورد - دليل التحديثات

## نظرة عامة

تم إعادة تصميم صفحة الداشبورد الرئيسية لتكون مقسمة عمودياً، مع قسم للمانجا وقسم للروايات، مما يسهل على المستخدمين الوصول إلى المحتوى المفضل لديهم.

## التغييرات الرئيسية

### 1. إعادة تسمية الصفحات
- **dashboard.html** → **manga.html**: أصبحت صفحة مخصصة للمانجا فقط
- **dashboard.html** (جديد): صفحة رئيسية مقسمة عمودياً

### 2. الهيكل الجديد للداشبورد

```
الداشبورد الرئيسي (dashboard.html)
├── عمود المانجا (يسار)
│   ├── إحصائيات المانجا
│   ├── آخر تحديثات المانجا
│   ├── الأكثر شعبية
│   └── أزرار الوصول السريع
└── عمود الروايات (يمين)
    ├── إحصائيات الروايات
    ├── آخر تحديثات الروايات
    ├── الأكثر شعبية
    └── أزرار الوصول السريع
```

### 3. الصفحات المتاحة

| الصفحة | الوصف | المسار |
|--------|-------|--------|
| الداشبورد الرئيسي | صفحة مقسمة للمانجا والروايات | `/dashboard.html` |
| صفحة المانجا | صفحة مخصصة للمانجا فقط | `/manga.html` |
| صفحة الروايات | صفحة مخصصة للروايات فقط | `/novels.html` |
| المكتبة | مكتبة شاملة للمانجا | `/library.html` |
| المفضلة | قائمة المفضلة | `/favorites.html` |
| السجل | سجل المشاهدة | `/history.html` |

### 4. شريط التنقل (Navbar)

تم تحديث شريط التنقل ليشمل:
- الرئيسية (dashboard.html)
- المانجا (manga.html)
- الروايات (novels.html)
- المكتبة (library.html)
- المفضلة (favorites.html)
- السجل (history.html)
- ديسكورد

## التصميم responsive

الداشبورد الجديد متجاوب مع جميع الأحجام:
- **الشاشات الكبيرة (>1024px)**: عمودين متوازيين
- **الأجهزة اللوحية (<1024px)**: عمود واحد
- **الهواتف (<600px)**: عمود واحد مع تكبير العناصر

## الألوان والهوية البصرية

- **المانجا**: اللون الأساسي (primary-color) - أزرق/بنفسجي
- **الروايات**: اللون الثانوي (secondary-color) - برتقالي/ذهبي

## كيفية التشغيل

1. تأكد من تشغيل قاعدة البيانات:
```bash
docker-compose up -d
```

2. قم بترحيل جداول الروايات:
```bash
psql -U postgres -d msai_db -f scripts/novel_tables.sql
```

3. شغل التطبيق:
```bash
cd MS-AI
go run cmd/web/main.go
```

4. افتح المتصفح:
- الداشبورد: `http://localhost:8080/dashboard.html`
- المانجا: `http://localhost:8080/manga.html`
- الروايات: `http://localhost:8080/novels.html`

## التحديثات المستقبلية

1. **إشعارات مخصصة**: إشعارات عند إضافة فصول جديدة للمانجا أو الروايات المفضلة
2. **توصيات ذكية**: توصيات مبنية على سجل المشاهدة
3. **إحصائيات متقدمة**: رسوم بيانية لتطور المشاهدة
4. **وضع القراءة الليلي**: تحسين تجربة القراءة في الظلام
